## ADDED Requirements

### Requirement: Homepage structure
Die Website SHALL eine klare Struktur mit den folgenden Bereichen haben:
- Über mich
- Was bisher geschah
- Was die Zukunft bringt
- Impressum

#### Scenario: Navigation zwischen Bereichen
- **WHEN** der Benutzer auf einen Bereich in der Navigation klickt
- **THEN** wird der Benutzer zu dem entsprechenden Abschnitt der Website weitergeleitet.

### Requirement: Project gallery
Die Website SHALL Bildergalerien für abgeschlossene und potenzielle Projekte bereitstellen.

#### Scenario: Anzeige abgeschlossener Projekte
- **WHEN** der Benutzer den Bereich "Was bisher geschah" öffnet
- **THEN** werden Bilder der abgeschlossenen Projekte angezeigt.

#### Scenario: Anzeige potenzieller Projekte
- **WHEN** der Benutzer den Bereich "Was die Zukunft bringt" öffnet
- **THEN** werden Bilder der potenziellen Projekte angezeigt.
